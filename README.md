# Diploma Notes - Educational Website

A comprehensive educational platform for polytechnic students providing notes, practicals, video tutorials, and previous year questions (PYQs) for Electrical, Mechanical, and Chemical Engineering branches.

## 🎯 Features

- **Mobile-First Design** - Optimized for mobile devices with responsive layout
- **Dynamic Content** - JSON-based content management
- **Recently Viewed** - Automatic tracking of user's study progress
- **Lazy Loading** - Optimized performance with lazy image loading
- **Clean Architecture** - Well-organized file structure with reusable components
- **localStorage Integration** - Remembers user preferences and history
